# 🚀 Premium Kanban Board

A state-of-the-art, responsive Kanban board built with **React**, **TypeScript**, and **Zustand**. This project was designed to meet strict UI/UX and architectural requirements for a high-end frontend assignment.

## ✨ Features

- **Dynamic Columns**: Todo, In Progress, and Done.
- **Drag & Drop**: Seamless card movement with `@dnd-kit`.
- **Inline Editing**: Sophisticated editing logic with a custom centered confirmation modal.
- **Persistent State**: Automatic saving and restoration via LocalStorage middleware.
- **Responsive Design**: Premium fluid layout that adapts from 4K desktops to mobile devices.
- **Real-time Notifications**: Premium dark-themed feedback for all actions.

## 🛠️ Technology Stack

- **Framework**: React 19 + TypeScript
- **Styling**: Vanilla CSS (Custom Variable Design System)
- **State Management**: Zustand (with Persist Middleware)
- **Drag and Drop**: @dnd-kit/core & @dnd-kit/sortable
- **Notifications**: react-hot-toast
- **Bundler**: Vite
